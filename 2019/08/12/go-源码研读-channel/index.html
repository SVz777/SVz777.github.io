功能实现

新建channel ch := make(chan int)

创建一个channel的时候Go的语法是ch := make(chan datatype),这个在底层的实现实际是makechan

向channel发送数据 ch <- 1

向一个chan发送数据的语法是chan <- data,在底层会被转为chansend1

从channel接收数据<- ch,a =<- ch,a,ok = <- ch

整体跟发送差不多，底层对应两种情况chanrecv1和chanrecv2

chanrecv1对应着<- ch,a =<- ch,赋值的操作还是在汇编做的

chanrecv2则对应着a,ok =<- ch

关闭channel close(ch)
